# Book Management System

一个基于 Spring MVC 的图书管理系统。

## 技术栈

- **Java** - 后端开发语言
- **Spring MVC** - Web 框架
- **Spring JDBC** - 数据库访问
- **MySQL** - 数据库

## 项目结构

```
src/
├── main/
│   ├── java/          # Java 源代码
│   └── resources/    # 配置文件和 SQL 脚本
└── test/             # 测试代码
```

## 功能

- 图书信息管理（增删改查）
- 用户借阅记录管理
- 图书分类管理

## 快速开始

1. 导入数据库脚本（`src/main/resources/` 目录下的 SQL 文件）
2. 修改数据库配置
3. 运行项目
